# Atividade prática individual - Tópico 05

## Objetivo

Monitorizar o sistema, analisar logs, criar backups e definir medidas de continuidade operacional.

## Ambiente utilizado

VM local Ubuntu 26.04 executada através do Virt-Manager sobre WSL2.

## Atividades realizadas

- Monitorização do sistema.
- Consulta de logs.
- Backup do serviço web.
- Recuperação em ambiente de teste.
- Definição de plano de continuidade.

## Evidências produzidas

- Monitorização.
- Logs.
- Backup.
- Recuperação.
- Continuidade operacional.

## Resultado

O sistema foi monitorizado com sucesso e o backup foi restaurado corretamente.
